# Installation

## GitHub
To get the latest version from Github, just run
`sudo pip(3) install -U git+git://github.com/TurtleP/eclipse.git`

## Local
1. Clone the repository:
  - `git clone https://github.com/TurtleP/eclipse`
2. Run pip install
  - `sudo pip(3) install -U eclipse`
